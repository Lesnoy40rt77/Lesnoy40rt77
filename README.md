## About me

I’m a backend-focused developer interested in **real products**, **infrastructure**, and the parts of software engineering that begin after a nice demo:

Inside the project I work on **backend/API**, **staging deployment**, **CI**, and the technical glue that keeps the product moving toward a real MVP.
